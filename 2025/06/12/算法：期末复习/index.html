动态规划

DP 的一般方法

  • 算法设计方法：
  1. 找到子问题（或从递归解法合并重复子问题）
  2. 确认在 DAG 中，并找到 DAG 的拓扑序
  3. 按照拓扑序，求解子问题并存储

DP 算法设计的关键：子问题定义 & 状态转移方程

  • 正确性证明：数学归纳法（按拓扑序）
  • 优先级队列优化：掌握类似维护 PLL 的基础方法即可

    • 初始化：设初始流 f(e)=0f(e) = 0f(e)=0，构建初始剩余网络 GfG_fGf​
    • 迭代过程：

      • ​计算瓶颈容量：找出路径 ppp 上残留容量的最小值 bbb

NP 完全

Karp 归约与证明 NP 完全

  • 常见的 NP 完全问题：

▶
SAT 问题

给定一个 CNF 表达式 ϕ\phiϕ，判断其是否存在使 ϕ\phiϕ 为真的赋值。

▶
3SAT 问题

给定一个 3-CNF 表达式 ϕ\phiϕ，判断其是否存在使 ϕ\phiϕ 为真的赋值。3-CNF 表达式是指每个字句只包含最多三个文字的 CNF。

▶
顶点覆盖问题

给定一个无向图 G=(V,E)G = (V, E)G=(V,E) 和正整数 kkk，判断是否 GGG 存在大小为 kkk 的顶点覆盖。对一个无向图 G=(V,E)G = (V, E)G=(V,E)，如果每条边都至少有一个端点在 S⊆VS \subseteq VS⊆V中，那么 SSS 是 GGG 的一个顶点覆盖。

▶
独立集问题

给定一个无向图 G=(V,E)G = (V, E)G=(V,E) 和正整数 kkk，判断是否 GGG 存在大小为 kkk 的独立集。对一个无向图 G=(V,E)G = (V, E)G=(V,E)，如果顶点集合 S⊆VS \subseteq VS⊆V 中任意两个顶点都不相邻，那么 SSS 是 GGG 中的一个独立集。

▶
团问题

给定一个无向图 G=(V,E)G = (V, E)G=(V,E) 和正整数 kkk，判断是否 GGG 存在大小为 kkk 的团。对一个无向图 G=(V,E)G = (V, E)G=(V,E)，如果顶点集合 S⊆VS \subseteq VS⊆V 中任意两个顶点都相邻，那么 SSS 是 GGG 中的一个团。

▶
支配集问题

给定一个无向图 G=(V,E)G = (V, E)G=(V,E) 和正整数 kkk，判断是否 GGG 存在大小为 kkk 的支配集。对一个无向图 G=(V,E)G = (V, E)G=(V,E) 和顶点集合 S⊆VS \subseteq VS⊆V，如果每个不在 SSS 中的顶点都存在 SSS 中的顶点与它相邻，那么 SSS 是 GGG 的一个支配集。

▶
子集和问题

给定一些整数 S={a1,…,an}S = \{a_1, \dots, a_n\}S={a1​,…,an​} 和正整数 kkk，判断是否存在 T⊆ST \subseteq ST⊆S 使得 ∑ai∈Tai=k\sum_{a_i \in T} a_i = k∑ai​∈T​ai​=k。

▶
划分问题

给定一些整数 S={a1,…,an}S = \{a_1, \dots, a_n\}S={a1​,…,an​} 和正整数 kkk，判断是否能将 SSS 划分为 AAA 和 BBB 使得 ∑a∈Aa=∑b∈Bb\sum_{a \in A} a = \sum_{b \in B} b∑a∈A​a=∑b∈B​b。

▶
哈密顿路径问题

给定一个无向图 G=(V,E)G = (V, E)G=(V,E)，判断 GGG 是否存在哈密顿路径。对一个无向图 G=(V,E)G = (V, E)G=(V,E)，哈密顿路径指一条包含每个顶点恰好一次的路径。

  • 证明 NP：问题的正确实例（yes instance）能在多项式时间内被验证
  • 证明 NP 完全的步骤：
    1. 证明 fff 是 NP 问题
    2. 对一个 NP 完全问题 ggg，给出 g≤kfg \le_k fg≤k​f 的归约过程
      • 这表明 fff 的难度略高于 ggg
    3. 证明 ggg 的正确实例映射到 fff 的正确实例
    4. 证明 ggg 的错误实例映射到 fff 的错误实例
      • 或证明逆否命题，即映射到 fff 的正确实例的 ggg 实例都是正确的
