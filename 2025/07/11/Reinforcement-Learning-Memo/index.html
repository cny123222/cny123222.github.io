Reinforcement Learning Memo

Last updated on August 25, 2025 am

A simple memo for learning Reinforcement Learning.（强化学习备忘录）

Terminologies

  1. State 状态 sss, Action 动作 aaa, Reward 奖励 rrr
  2. Policy 策略 π:(s,a)→[0,1]\pi: (s,a) \to [0, 1]π:(s,a)→[0,1]

    π(s∣a)=P(A=a∣S=s)\pi(s | a) = \mathbb{P}(A = a | S = s) π(s∣a)=P(A=a∣S=s)

  3. Trajectory 轨迹: s1,a1,r1,s2,a2,r2,…,sT,aT,rTs_1, a_1, r_1, s_2, a_2, r_2, \dots, s_T, a_T, r_Ts1​,a1​,r1​,s2​,a2​,r2​,…,sT​,aT​,rT​
  4. Discounted return 折扣回报

    Ut=Rt+γRt+1+γ2Rt+2+γ3Rt+3+⋯U_t = R_t + \gamma R_{t + 1} + \gamma^2 R_{t + 2} + \gamma^3 R_{t + 3} + \cdots Ut​=Rt​+γRt+1​+γ2Rt+2​+γ3Rt+3​+⋯

  5. Action-value function 动作价值函数

    Qπ(st,at)=E[Ut∣S=st,A=at]Q_\pi(s_t,a_t) = \mathbb{E}[U_t | S = s_t, A = a_t] Qπ​(st​,at​)=E[Ut​∣S=st​,A=at​]

  6. Optimal action-value function 最优动作价值函数

    Q∗(st,at)=max⁡πQπ(st,at)Q^*(s_t,a_t) = \max_\pi Q_\pi(s_t,a_t) Q∗(st​,at​)=πmax​Qπ​(st​,at​)

  7. State-value function 状态价值函数

    Vπ(st)=EA[Qπ(st,A)]=∑a∈Aπ(a∣st)⋅Qπ(st,a)V_\pi(s_t) = \mathbb{E}_A[Q_\pi(s_t, A)] = \sum_{a \in A} \pi(a|s_t) \cdot Q_\pi(s_t, a) Vπ​(st​)=EA​[Qπ​(st​,A)]=a∈A∑​π(a∣st​)⋅Qπ​(st​,a)

    • ES[Vπ(S)]\mathbb{E}_S[V_\pi(S)]ES​[Vπ​(S)] evaluates how good the policy π\piπ is.

Value-based Learning

  • Goal: Approximate the Q function to maximize the total reward.

Temporal Difference (TD) Learning

Q(st,at;w)=rt+γ⋅Q(st+1,at+1;w)Q(s_t,a_t;\mathbf{w}) = r_t + \gamma \cdot Q(s_{t+1},a_{t+1};\mathbf{w}) Q(st​,at​;w)=rt​+γ⋅Q(st+1​,at+1​;w)

  • Prediction: Q(st,at;wt)Q(s_t,a_t;\mathbf{w}_t)Q(st​,at​;wt​)
  • TD target:

yt=rt+γ⋅Q(st+1,at+1;wt)y_t = r_t + \gamma \cdot Q(s_{t+1},a_{t+1};\mathbf{w}_t) yt​=rt​+γ⋅Q(st+1​,at+1​;wt​)

  • Loss:

Lt=12[Q(st,at;w)−yt]2L_t = \frac{1}{2}[Q(s_t,a_t;\mathbf{w}) - y_t]^2 Lt​=21​[Q(st​,at​;w)−yt​]2

  • Gradient descent:

wt+1=wt−α⋅∂Lt∂w∣w=wt\mathbf{w}_{t+1} = \mathbf{w}_t - \alpha \cdot \frac{\partial L_t}{\partial \mathbf{w}} \vert_{\mathbf{w} = \mathbf{w}_t} wt+1​=wt​−α⋅∂w∂Lt​​∣w=wt​​

SARSA

  • Goal: Learn the action-value function QπQ_\piQπ​.

Tabular Version

  • Goal: Directly learn Qπ(s,a)Q_\pi(s,a)Qπ​(s,a).
  • Algorithm:
    1. Observe a transition (st,at,rt,st+1)(s_t, a_t, r_t, s_{t+1})(st​,at​,rt​,st+1​)
    2. Sample at+1∼π(⋅∣st+1)a_{t+1} \sim \pi(\cdot|s_{t+1})at+1​∼π(⋅∣st+1​)
    3. TD target: yt=rt+γ⋅Qπ(st+1,at+1)y_t = r_t + \gamma \cdot Q_\pi(s_{t+1}, a_{t+1})yt​=rt​+γ⋅Qπ​(st+1​,at+1​)
    4. TD error: δt=Qπ(st,at)−yt\delta_t = Q_\pi(s_t,a_t) - y_tδt​=Qπ​(st​,at​)−yt​
    5. Update: Qπ(st,at)←Qπ(st,at)−α⋅δtQ_\pi(s_t,a_t) \gets Q_\pi(s_t,a_t) - \alpha \cdot \delta_tQπ​(st​,at​)←Qπ​(st​,at​)−α⋅δt​

Neural Network Version

  • Goal: Approximate Qπ(s,a)Q_\pi(s,a)Qπ​(s,a) by the value network q(s,a;w)q(s,a;\mathbf{w})q(s,a;w).
  • Algorithm:
    1. Observe a transition (st,at,rt,st+1)(s_t, a_t, r_t, s_{t+1})(st​,at​,rt​,st+1​)
    2. TD target: yt=rt+γ⋅q(st+1,at+1;w)y_t = r_t + \gamma \cdot q(s_{t+1},a_{t+1};\mathbf{w})yt​=rt​+γ⋅q(st+1​,at+1​;w)
    3. TD error: δt=q(st,at;w)−yt\delta_t = q(s_t,a_t;\mathbf{w}) - y_tδt​=q(st​,at​;w)−yt​
    4. Update: w←w−α⋅δt⋅∂q(st,at;w)∂w\mathbf{w} \gets \mathbf{w} - \alpha \cdot \delta_t \cdot \frac{\partial q(s_t,a_t;\mathbf{w})}{\partial \mathbf{w}}w←w−α⋅δt​⋅∂w∂q(st​,at​;w)​

Q-Learning

  • Goal: Learn the optimal action-value function Q∗Q^*Q∗.

Tabular Version

  • Goal: Directly learn Q∗(s,a)Q^*(s,a)Q∗(s,a).
  • Algorithm:
    1. Observe a transition (st,at,rt,st+1)(s_t, a_t, r_t, s_{t+1})(st​,at​,rt​,st+1​)
    2. TD target: yt=rt+γ⋅max⁡aQ∗(st+1,a)y_t = r_t + \gamma \cdot \max_a Q^*(s_{t+1}, a)yt​=rt​+γ⋅maxa​Q∗(st+1​,a)
    3. TD error: δt=Q∗(st,at)−yt\delta_t = Q^*(s_t,a_t) - y_tδt​=Q∗(st​,at​)−yt​
    4. Update: Q∗(st,at)←Q∗(st,at)−α⋅δtQ^*(s_t,a_t) \gets Q^*(s_t,a_t) - \alpha \cdot \delta_tQ∗(st​,at​)←Q∗(st​,at​)−α⋅δt​

DQN Version

  • Goal: Approximate Q∗(s,a)Q^*(s,a)Q∗(s,a) by the DQN, Q(s,a;w)Q(s,a;\mathbf{w})Q(s,a;w)
  • Policy: Choose at=arg max⁡aQ(st,a;w)a_t = \argmax_{a} Q(s_t,a;\mathbf{w})at​=argmaxa​Q(st​,a;w)
  • Algorithm:
    1. Observe a transition (st,at,rt,st+1)(s_t, a_t, r_t, s_{t+1})(st​,at​,rt​,st+1​)
    2. TD target: yt=rt+γ⋅max⁡aQ(st+1,a;w)y_t = r_t + \gamma \cdot \max_a Q(s_{t+1}, a; \mathbf{w})yt​=rt​+γ⋅maxa​Q(st+1​,a;w)
    3. TD error: δt=Q(st,at;w)−yt\delta_t = Q(s_t,a_t;\mathbf{w}) - y_tδt​=Q(st​,at​;w)−yt​
    4. Update: w←w−α⋅δt⋅∂Q(st,at;w)∂w\mathbf{w} \gets \mathbf{w} - \alpha \cdot \delta_t \cdot \frac{\partial Q(s_t,a_t;\mathbf{w})}{\partial \mathbf{w}}w←w−α⋅δt​⋅∂w∂Q(st​,at​;w)​

Policy-based Learning

  • Goal: Learn θ\mathbf{\theta}θ that maximize J(θ)=ES[V(S;θ)]J(\mathbf{\theta}) = \mathbb{E}_S[V(S;\mathbf{\theta})]J(θ)=ES​[V(S;θ)].
  • Idea: Approximate policy function π(a∣s)\pi(a|s)π(a∣s) by policy network π(a∣s;θ)\pi(a|s;\mathbf{\theta})π(a∣s;θ).
    • Learn the policy network by policy gradient.

Policy Gradient

  • Policy gradient:

∂V(st;θ)∂θ=EAt∼π[∂ln⁡π(At∣st;θ)∂θ⋅Qπ(st,At)]\frac{\partial V(s_t;\mathbf{\theta})}{\partial \mathbf{\theta}} = \mathbb{E}_{A_t \sim \pi}\left[\frac{\partial \ln \pi(A_t|s_t;\mathbf{\theta})}{\partial \mathbf{\theta}} \cdot Q_{\pi}(s_t,A_t)\right] ∂θ∂V(st​;θ)​=EAt​∼π​[∂θ∂lnπ(At​∣st​;θ)​⋅Qπ​(st​,At​)]

  • Policy gradient with baseline: Suppose bbb is independent of AtA_tAt​, then

∂V(st;θ)∂θ= EAt∼π[∂ln⁡π(At∣st;θ)∂θ⋅(Qπ(st,At)−b)]= EAt∼π[∂ln⁡π(At∣st;θ)∂θ⋅(Qπ(st,At)−Vπ(st))]\begin{aligned} \frac{\partial V(s_t;\mathbf{\theta})}{\partial \mathbf{\theta}} =& \, \mathbb{E}_{A_t \sim \pi}\left[\frac{\partial \ln \pi(A_t|s_t;\mathbf{\theta})}{\partial \mathbf{\theta}} \cdot (Q_{\pi}(s_t,A_t) - b)\right] \\ =& \, \mathbb{E}_{A_t \sim \pi}\left[\frac{\partial \ln \pi(A_t|s_t;\mathbf{\theta})}{\partial \mathbf{\theta}} \cdot \left(Q_{\pi}(s_t,A_t) - V_\pi(s_t)\right)\right] \end{aligned} ∂θ∂V(st​;θ)​==​EAt​∼π​[∂θ∂lnπ(At​∣st​;θ)​⋅(Qπ​(st​,At​)−b)]EAt​∼π​[∂θ∂lnπ(At​∣st​;θ)​⋅(Qπ​(st​,At​)−Vπ​(st​))]​

REINFORCE

  • Goal: Approximate Qπ(st,At)Q_{\pi}(s_t,A_t)Qπ​(st​,At​) by utu_tut​ and Vπ(st)V_\pi(s_t)Vπ​(st​) by value network v(st;w)v(s_t;\mathbf{w})v(st​;w)
  • Algorithm:
    1. Play a game to the end and observe the trajectory:

    s1,a1,r1,s2,a2,r2,…,sn,an,rns_1, a_1, r_1, s_2, a_2, r_2, \ldots, s_n, a_n, r_n s1​,a1​,r1​,s2​,a2​,r2​,…,sn​,an​,rn​

    1. Compute ut=∑i=tnγr−t⋅riu_t = \sum_{i=t}^n \gamma^{r-t}\cdot r_iut​=∑i=tn​γr−t⋅ri​ and δt=v(st;w)−ut\delta_t = v(s_t;\mathbf{w}) - u_tδt​=v(st​;w)−ut​
    2. Update the policy network by:

    θ←θ−β⋅δt⋅∂ln⁡π(at∣st;θ)∂θ\mathbf{\theta} \gets \mathbf{\theta} - \beta \cdot \delta_t \cdot \frac{\partial \ln \pi(a_t|s_t;\mathbf{\theta})}{\partial \mathbf{\theta}} θ←θ−β⋅δt​⋅∂θ∂lnπ(at​∣st​;θ)​

    1. Update the value network by:

    w←w−α⋅δt⋅∂v(st;w)∂w\mathbf{w} \gets \mathbf{w} - \alpha \cdot \delta_t \cdot \frac{\partial v(s_t;\mathbf{w})}{\partial \mathbf{w}} w←w−α⋅δt​⋅∂w∂v(st​;w)​

    1. Repeat procedure 2 to 4 for t=1,…,nt = 1, \dots, nt=1,…,n

Actor-Critic

  • Goal: Approximate policy function π(a∣s)\pi(a|s)π(a∣s) by policy network π(a∣s;θ)\pi(a|s;\mathbf{\theta})π(a∣s;θ) and state-value function Vπ(s)V_\pi(s)Vπ​(s) by value network v(s;w)v(s;\mathbf{w})v(s;w).
    • Actor: Update policy network π(a∣s;θ)\pi(a|s;\mathbf{\theta})π(a∣s;θ) using policy gradient to increase V(s;θ,w)V(s;\mathbf{\theta}, \mathbf{w})V(s;θ,w)
    • Critic: Update value network v(s;w)v(s;\mathbf{w})v(s;w) using TD learning to better estimate the return
  • Algorithm:
    1. Observe a transition (st,at,rt,st+1)(s_t,a_t,r_t,s_{t+1})(st​,at​,rt​,st+1​)
    2. TD target: yt=rt+γ⋅v(st+1;w)y_t = r_t + \gamma \cdot v(s_{t+1};\mathbf{w})yt​=rt​+γ⋅v(st+1​;w)
    3. TD error: δt=v(st;w)−yt\delta_t = v(s_t;\mathbf{w}) - y_tδt​=v(st​;w)−yt​
    4. Update the policy network (actor) by:

    θ←θ−β⋅δt⋅∂ln⁡π(at∣st;θ)∂θ\mathbf{\theta} \gets \mathbf{\theta} - \beta \cdot \delta_t \cdot \frac{\partial \ln \pi(a_t|s_t;\mathbf{\theta})}{\partial \mathbf{\theta}} θ←θ−β⋅δt​⋅∂θ∂lnπ(at​∣st​;θ)​

    1. Update the value network (critic) by:

    w←w−α⋅δt⋅∂v(st;w)∂w\mathbf{w} \gets \mathbf{w} - \alpha \cdot \delta_t \cdot \frac{\partial v(s_t;\mathbf{w})}{\partial \mathbf{w}} w←w−α⋅δt​⋅∂w∂v(st​;w)​

TRPO (Trust Region Policy Optimization)

  • Trust region algorithm: Find θ∗=arg max⁡θJ(θ)\mathbf{\theta}^* = \argmax_{\mathbf{\theta}} J(\mathbf{\theta})θ∗=argmaxθ​J(θ).
    1. Approximation: Given θold\mathbf{\theta}_\text{old}θold​, construct L(θ∣θold)L(\mathbf{\theta}|\mathbf{\theta}_\text{old})L(θ∣θold​), which is an approximation to J(θ)J(\mathbf{\theta})J(θ) in N(θold)\mathcal{N}(\mathbf{\theta}_\text{old})N(θold​)
    2. Maximization: In the trust region, find θnew\mathbf{\theta}_\text{new}θnew​ by:

    θnew←arg max⁡θ∈N(θold)L(θ∣θold)\mathbf{\theta}_\text{new} \gets \argmax_{\mathbf{\theta} \in \mathcal{N}(\mathbf{\theta}_\text{old})} L(\mathbf{\theta}|\mathbf{\theta}_\text{old}) θnew​←θ∈N(θold​)argmax​L(θ∣θold​)

  • Object function:

J(θ)=ES[Vπ(S)]=ES[EA∼π(⋅∣s;θold)[π(A∣S;θ)π(A∣S;θold)⋅Qπ(S,A)]]J(\mathbf{\theta}) = \mathbb{E}_S[V_\pi(S)] = \mathbb{E}_S\left[\mathbb{E}_{A \sim \pi(\cdot|s;\mathbf{\theta}_\text{old})}\left[\frac{\pi(A|S;\mathbf{\theta})}{\pi(A|S;\mathbf{\theta}_\text{old})} \cdot Q_\pi(S,A)\right]\right] J(θ)=ES​[Vπ​(S)]=ES​[EA∼π(⋅∣s;θold​)​[π(A∣S;θold​)π(A∣S;θ)​⋅Qπ​(S,A)]]

  • Algorithm:
    1. Controlled by the policy π(⋅∣s;θold)\pi(\cdot|s;\mathbf{\theta}_\text{old})π(⋅∣s;θold​), the agent plays a game to the end and observes a trajectory:

    s1,a1,r1,s2,a2,r2,…,sn,an,rns_1, a_1, r_1, s_2, a_2, r_2, \ldots, s_n, a_n, r_n s1​,a1​,r1​,s2​,a2​,r2​,…,sn​,an​,rn​

    1. For i=1,2,…,ni = 1, 2, \ldots, ni=1,2,…,n, compute discounted returns: ui=∑k=inγk−i⋅rku_i = \sum_{k=i}^n \gamma^{k-i} \cdot r_kui​=∑k=in​γk−i⋅rk​
    2. Approximation:

    L~(θ∣θold)=1n∑i=1nπ(ai∣si;θ)π(ai∣si;θold)⋅ui\tilde{L}(\mathbf{\theta}|\mathbf{\theta}_\text{old}) = \frac{1}{n} \sum_{i=1}^n \frac{\pi(a_i|s_i;\mathbf{\theta})}{\pi(a_i|s_i;\mathbf{\theta}_\text{old})} \cdot u_i L~(θ∣θold​)=n1​i=1∑n​π(ai​∣si​;θold​)π(ai​∣si​;θ)​⋅ui​

    1. Maximization:

    θnew←arg max⁡θL~(θ∣θold);s.t. ∥θ−θold∥<Δ\mathbf{\theta}_\text{new} \gets \argmax_{\mathbf{\theta}} \tilde{L}(\mathbf{\theta}|\mathbf{\theta}_\text{old}); \quad s.t. \, \Vert \mathbf{\theta} - \mathbf{\theta}_\text{old} \Vert < \Delta θnew​←θargmax​L~(θ∣θold​);s.t.∥θ−θold​∥<Δ

  • Goal:

    arg max⁡θ′Es∼vθ,a∼πθ(⋅∣s)[πθ′(a,s)πθ(a,s)⋅Aπθ(s,a)]s.t.DKL(πθ(⋅∣s)∥πθ′(⋅∣s))<Δ\begin{aligned} & \argmax_{\mathbf{\theta'}} \mathbb{E}_{s \sim v_{\mathbf{\theta}}, a \sim \pi_{\mathbf{\theta}(\cdot|s)}} \left[\frac{\pi_{\mathbf{\theta'}}(a,s)}{\pi_{\mathbf{\theta}}(a,s)} \cdot A_{\pi_{\mathbf{\theta}}}(s,a)\right] \\ s.t. & \quad D_{KL}(\pi_{\mathbf{\theta}}(\cdot|s) \Vert \pi_{\mathbf{\theta'}}(\cdot|s)) < \Delta \end{aligned} s.t.​θ′argmax​Es∼vθ​,a∼πθ(⋅∣s)​​[πθ​(a,s)πθ′​(a,s)​⋅Aπθ​​(s,a)]DKL​(πθ​(⋅∣s)∥πθ′​(⋅∣s))<Δ​

    where Aπθ(s,a)A_{\pi_{\mathbf{\theta}}}(s,a)Aπθ​​(s,a) is the advantage function.

PPO (Proximal Policy Optimization)

  • PPO-penalty:

    arg max⁡θ′Es∼vθ,a∼πθ(⋅∣s)[πθ′(a,s)πθ(a,s)⋅AπθGAE(s,a)−β⋅DKL(πθ(⋅∣s)∥πθ′(⋅∣s))]\argmax_{\mathbf{\theta'}} \mathbb{E}_{s \sim v_{\mathbf{\theta}}, a \sim \pi_{\mathbf{\theta}(\cdot|s)}} \left[\frac{\pi_{\mathbf{\theta'}}(a,s)}{\pi_{\mathbf{\theta}}(a,s)} \cdot A^{GAE}_{\pi_{\mathbf{\theta}}}(s,a) - \beta \cdot D_{KL}(\pi_{\mathbf{\theta}}(\cdot|s) \Vert \pi_{\mathbf{\theta'}}(\cdot|s))\right] θ′argmax​Es∼vθ​,a∼πθ(⋅∣s)​​[πθ​(a,s)πθ′​(a,s)​⋅Aπθ​GAE​(s,a)−β⋅DKL​(πθ​(⋅∣s)∥πθ′​(⋅∣s))]

    • β←β/2if DKL<δ/1.5\beta \gets \beta / 2 \quad \text{if } D_{KL} < \delta / 1.5β←β/2if DKL​<δ/1.5
    • β←β×2if DKL>δ×1.5\beta \gets \beta \times 2 \quad \text{if } D_{KL} > \delta \times 1.5β←β×2if DKL​>δ×1.5
  • PPO-clip:

    arg max⁡θ′Es∼vθ,a∼πθ(⋅∣s)[min⁡(πθ′(a,s)πθ(a,s)AπθGAE(s,a),clip(πθ′(a,s)πθ(a,s),1−ϵ,1+ϵ)AπθGAE(s,a))]\argmax_{\mathbf{\theta'}} \mathbb{E}_{s \sim v_{\mathbf{\theta}}, a \sim \pi_{\mathbf{\theta}(\cdot|s)}} \left[\min\left(\frac{\pi_{\mathbf{\theta'}}(a,s)}{\pi_{\mathbf{\theta}}(a,s)}A^{GAE}_{\pi_{\mathbf{\theta}}}(s,a), clip\left(\frac{\pi_{\mathbf{\theta'}}(a,s)}{\pi_{\mathbf{\theta}}(a,s)}, 1-\epsilon, 1+\epsilon\right)A^{GAE}_{\pi_{\mathbf{\theta}}}(s,a)\right)\right] θ′argmax​Es∼vθ​,a∼πθ(⋅∣s)​​[min(πθ​(a,s)πθ′​(a,s)​Aπθ​GAE​(s,a),clip(πθ​(a,s)πθ′​(a,s)​,1−ϵ,1+ϵ)Aπθ​GAE​(s,a))]
