Reinforcement Learning Memo

Last updated on July 15, 2025 pm

A simple memo for learning Reinforcement Learning.（强化学习备忘录）

Terminologies

  1. State 状态 sss, Action 动作 aaa, Reward 奖励 rrr
  2. Policy 策略 π:(s,a)→[0,1]\pi: (s,a) \to [0, 1]π:(s,a)→[0,1]

    π(s∣a)=P(A=a∣S=s)\pi(s | a) = \mathbb{P}(A = a | S = s) π(s∣a)=P(A=a∣S=s)

  3. Trajectory 轨迹: s1,a1,r1,s2,a2,r2,…,sT,aT,rTs_1, a_1, r_1, s_2, a_2, r_2, \dots, s_T, a_T, r_Ts1​,a1​,r1​,s2​,a2​,r2​,…,sT​,aT​,rT​
  4. Discounted return 折扣回报

    Ut=Rt+γRt+1+γ2Rt+2+γ3Rt+3+⋯U_t = R_t + \gamma R_{t + 1} + \gamma^2 R_{t + 2} + \gamma^3 R_{t + 3} + \cdots Ut​=Rt​+γRt+1​+γ2Rt+2​+γ3Rt+3​+⋯

  5. Action-value function 动作价值函数

    Qπ(st,at)=E[Ut∣S=st,A=at]Q_\pi(s_t,a_t) = \mathbb{E}[U_t | S = s_t, A = a_t] Qπ​(st​,at​)=E[Ut​∣S=st​,A=at​]

  6. Optimal action-value function 最优动作价值函数

    Q∗(st,at)=max⁡πQπ(st,at)Q^*(s_t,a_t) = \max_\pi Q_\pi(s_t,a_t) Q∗(st​,at​)=πmax​Qπ​(st​,at​)

  7. State-value function 状态价值函数

    Vπ(st)=EA[Qπ(st,A)]=∑a∈Aπ(a∣st)⋅Qπ(st,a)V_\pi(s_t) = \mathbb{E}_A[Q_\pi(s_t, A)] = \sum_{a \in A} \pi(a|s_t) \cdot Q_\pi(s_t, a) Vπ​(st​)=EA​[Qπ​(st​,A)]=a∈A∑​π(a∣st​)⋅Qπ​(st​,a)

    • ES[Vπ(S)]\mathbb{E}_S[V_\pi(S)]ES​[Vπ​(S)] evaluates how good the policy π\piπ is.

Value-based Learning

  • Goal: Approximate the Q function to maximize the total reward.

Temporal Difference (TD) Learning

Q(st,at;w)=rt+γ⋅Q(st+1,at+1;w)Q(s_t,a_t;\mathbf{w}) = r_t + \gamma \cdot Q(s_{t+1},a_{t+1};\mathbf{w}) Q(st​,at​;w)=rt​+γ⋅Q(st+1​,at+1​;w)

  • Prediction: Q(st,at;wt)Q(s_t,a_t;\mathbf{w}_t)Q(st​,at​;wt​)
  • TD target:

yt= rt+γ⋅Q(st+1,at+1;wt)= rt+γ⋅max⁡aQ(st+1,a;wt)\begin{aligned} y_t =& \, r_t + \gamma \cdot Q(s_{t+1},a_{t+1};\mathbf{w}_t) \\ =& \, r_t + \gamma \cdot \max_{a} Q(s_{t+1},a;\mathbf{w}_t) \end{aligned} yt​==​rt​+γ⋅Q(st+1​,at+1​;wt​)rt​+γ⋅amax​Q(st+1​,a;wt​)​

  • Loss:

Lt=12[Q(st,at;w)−yt]2L_t = \frac{1}{2}[Q(s_t,a_t;\mathbf{w}) - y_t]^2 Lt​=21​[Q(st​,at​;w)−yt​]2

  • Gradient descent:

wt+1=wt−α⋅∂Lt∂w∣w=wt\mathbf{w}_{t+1} = \mathbf{w}_t - \alpha \cdot \frac{\partial L_t}{\partial \mathbf{w}} \vert_{\mathbf{w} = \mathbf{w}_t} wt+1​=wt​−α⋅∂w∂Lt​​∣w=wt​​

  • Algorithm: one iteration of TD learning
    1. Observe state St=stS_t = s_tSt​=st​ and action At=atA_t = a_tAt​=at​
    2. Predict the value: qt=Q(st,at;wt)q_t = Q(s_t,a_t;\mathbf{w}_t)qt​=Q(st​,at​;wt​)
    3. Differentiate the value network: dt=∂Q(st,at;w)∂w∣w=wt\mathbf{d}_t = \frac{\partial Q(s_t,a_t;\mathbf{w})}{\partial \mathbf{w}} \vert_{\mathbf{w} = \mathbf{w}_t}dt​=∂w∂Q(st​,at​;w)​∣w=wt​​
    4. Environment provides new state st+1s_{t+1}st+1​ and reward rtr_trt​
    5. Compute TD target: yt=rt+γ⋅max⁡aQ(st+1,a;wt)y_t = r_t + \gamma \cdot \max_{a} Q(s_{t+1},a;\mathbf{w}_t)yt​=rt​+γ⋅maxa​Q(st+1​,a;wt​)
    6. Gradient descent: wt+1=wt−α⋅(qt−yt)⋅dt\mathbf{w}_{t+1} = \mathbf{w}_t - \alpha \cdot (q_t - y_t) \cdot \mathbf{d}_twt+1​=wt​−α⋅(qt​−yt​)⋅dt​

SARSA

Q-Learning

Deep Q Network (DQN)

  • Idea: Approximate Q∗(s,a)Q^*(s,a)Q∗(s,a) using a neural network.
  • Network: Q(s,a;w)Q(s,a;\mathbf{w})Q(s,a;w)
    • Input: observed state sss
    • Output: scores for every action a∈Aa \in \mathcal{A}a∈A
  • Policy: Choose at=arg max⁡aQ(st,a;w)a_t = \argmax_{a} Q(s_t,a;\mathbf{w})at​=argmaxa​Q(st​,a;w)

Policy-based Learning

  • Goal: Learn θ\mathbf{\theta}θ that maximize J(θ)=ES[V(S;θ)]J(\mathbf{\theta}) = \mathbb{E}_S[V(S;\mathbf{\theta})]J(θ)=ES​[V(S;θ)].
  • Idea: Approximate policy function π(a∣s)\pi(a|s)π(a∣s) by policy network π(a∣s;θ)\pi(a|s;\mathbf{\theta})π(a∣s;θ).
    • Learn the policy network by policy gradient.

Policy Gradient

  • Form 1:

∂V(s;θ)∂θ=∑a∂π(a∣s;θ)∂θ⋅Qπ(s,a)\frac{\partial V(s;\mathbf{\theta})}{\partial \mathbf{\theta}} = \sum_{a} \frac{\partial \pi(a|s;\mathbf{\theta})}{\partial \mathbf{\theta}} \cdot Q_{\pi}(s,a) ∂θ∂V(s;θ)​=a∑​∂θ∂π(a∣s;θ)​⋅Qπ​(s,a)

  • Form 2:

∂V(s;θ)∂θ=EA∼π(⋅∣s;θ)[∂log⁡π(A∣s;θ)∂θ⋅Qπ(s,A)]\frac{\partial V(s;\mathbf{\theta})}{\partial \mathbf{\theta}} = \mathbb{E}_{A \sim \pi(\cdot|s;\mathbf{\theta})}\left[\frac{\partial \log \pi(A|s;\mathbf{\theta})}{\partial \mathbf{\theta}} \cdot Q_{\pi}(s,A)\right] ∂θ∂V(s;θ)​=EA∼π(⋅∣s;θ)​[∂θ∂logπ(A∣s;θ)​⋅Qπ​(s,A)]

  • Algorithm:
    1. Observe the state sts_tst​
    2. Randomly sample action ata_tat​ according to π(⋅∣st;θt)\pi(\cdot|s_t;\mathbf{\theta}_t)π(⋅∣st​;θt​)
    3. Compute qt≈Qπ(st,at)q_t \approx Q_\pi(s_t,a_t)qt​≈Qπ​(st​,at​)
    4. Differentiate policy network: dθ,t=∂log⁡π(at∣st,θ)∂θ∣θ=θt\mathbf{d}_{\mathbf{\theta},t} = \frac{\partial \log \pi(a_t|s_t,\mathbf{\theta})}{\partial \mathbf{\theta}}|_{\mathbf{\theta} = \mathbf{\theta}_t}dθ,t​=∂θ∂logπ(at​∣st​,θ)​∣θ=θt​​
    5. (Approximate) policy gradient: g(at,θt)=qt⋅dθ,t\mathbf{g}(a_t, \mathbf{\theta}_t) = q_t \cdot \mathbf{d}_{\mathbf{\theta}, t}g(at​,θt​)=qt​⋅dθ,t​
    6. Update policy network: θt+1=θt+β⋅g(at,θt)\mathbf{\theta}_{t+1} = \mathbf{\theta}_t + \beta \cdot \mathbf{g}(a_t, \mathbf{\theta}_t)θt+1​=θt​+β⋅g(at​,θt​)

Actor-Critic

  • State-value function is approximated using nerual networks.

V(s;θ,w)=∑aπ(a∣s;θ)⋅q(s,a;w)V(s;\mathbf{\theta}, \mathbf{w}) = \sum_{a} \pi(a|s;\mathbf{\theta}) \cdot q(s,a;\mathbf{w}) V(s;θ,w)=a∑​π(a∣s;θ)⋅q(s,a;w)

  • Actor: Update policy network π(a∣s;θ)\pi(a|s;\mathbf{\theta})π(a∣s;θ) using policy gradient to increase V(s;θ,w)V(s;\mathbf{\theta}, \mathbf{w})V(s;θ,w)
  • Critic: Update value network q(s,a;w)q(s,a;\mathbf{w})q(s,a;w) using TD learning to better estimate the return
  • Algorithm:
    1. Observe state sts_tst​ and randomly sample at∼π(⋅∣st;θt)a_t \sim \pi(\cdot|s_t;\mathbf{\theta}_t)at​∼π(⋅∣st​;θt​)
    2. Perform ata_tat​; then the environment gives new state st+1s_{t+1}st+1​ and reward rtr_trt​
    3. Randomly sample a~t+1∼π(⋅∣st+1;θt)\tilde{a}_{t+1} \sim \pi(\cdot|s_{t+1};\mathbf{\theta}_t)a~t+1​∼π(⋅∣st+1​;θt​)
    4. Evaluate value network: qt=q(st,at;wt)q_t = q(s_t,a_t;\mathbf{w_t})qt​=q(st​,at​;wt​) and qt+1=q(st+1,a~t+1;wt)q_{t+1} = q(s_{t+1},\tilde{a}_{t+1};\mathbf{w_t})qt+1​=q(st+1​,a~t+1​;wt​)
    5. Compute TD error: δt=qt−(rt+γ⋅qt+1)\delta_t = q_t - (r_t + \gamma \cdot q_{t+1})δt​=qt​−(rt​+γ⋅qt+1​)
    6. Differentiate value network: dw,t=∂q(st,at;w)∂w∣w=wt\mathbf{d}_{\mathbf{w}, t} = \frac{\partial q(s_t,a_t;\mathbf{w})}{\partial \mathbf{w}}|_{\mathbf{w} = \mathbf{w}_t}dw,t​=∂w∂q(st​,at​;w)​∣w=wt​​
    7. Update value network: wt+1=wt−α⋅δt⋅dw,t\mathbf{w}_{t+1} = \mathbf{w}_t - \alpha \cdot \delta_t \cdot \mathbf{d}_{\mathbf{w}, t}wt+1​=wt​−α⋅δt​⋅dw,t​
    8. Differentiate policy network: dθ,t=∂log⁡π(at∣st;θ)∂θ∣w=wt\mathbf{d}_{\mathbf{\theta}, t} = \frac{\partial \log \pi(a_t|s_t;\mathbf{\theta})}{\partial \mathbf{\theta}}|_{\mathbf{w} = \mathbf{w}_t}dθ,t​=∂θ∂logπ(at​∣st​;θ)​∣w=wt​​
    9. Update policy network: θt+1=θt+β⋅qt⋅dθ,t\mathbf{\theta}_{t+1} = \mathbf{\theta}_t + \beta \cdot q_t \cdot \mathbf{d}_{\mathbf{\theta}, t}θt+1​=θt​+β⋅qt​⋅dθ,t​
