Python 使用 heapq 模块实现最小堆。

heapq 基础（最小堆）

Python 的 heapq 模块实现的是最小堆（堆顶元素最小），用一个普通列表来存储堆结构。

核心概念

  • 堆：特殊的完全二叉树，父节点 ≤ 子节点（最小堆）
  • 堆顶：列表的第 0 个元素 h[0]，总是当前最小值
  • 列表存储：堆用列表模拟二叉树，对于索引 i：
    • 左子节点索引 = 2*i + 1
    • 右子节点索引 = 2*i + 2
    • 父节点索引 = (i-1) // 2
